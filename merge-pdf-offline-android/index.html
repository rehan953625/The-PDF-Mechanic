Frequently Asked Questions

Why join PDF files offline?

Uploading private files to online servers can be risky. By using an offline tool like The PDF Mechanic, you ensure your documents (like Aadhaar, PAN, Marksheets) stay safe on your phone. Plus, it's faster because you don't have to upload/download heavy files.

Does this work on iPhone too?

Yes! While we optimized it for Android, it works perfectly on iPhone (iOS) using Safari or Chrome. It's fully compatible with all modern mobile browsers.

Is there a limit on file size?

Since the processing happens on your device, the only limit is your phone's memory (RAM). You can easily merge standard documents without any issues.
